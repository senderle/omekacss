CSS Generator

This allows you to generate CSS for the custom CSS plugin that is now available on Omeka.net. The customizations possible include color schemes, fonts, block borders, and column layout (not yet implemented) can all be adjusted. It assumes you're using the Seasons theme, with the "night" setting. Any other combination of theme and setting may not produce expected results.
